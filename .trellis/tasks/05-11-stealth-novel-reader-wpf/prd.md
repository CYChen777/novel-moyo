# Stealth Novel Reader - WPF摸鱼小说阅读器

## Goal

开发一款 Windows 桌面端"摸鱼"小说阅读器，通过背景透明、无任务栏、鼠标穿透等特性，让用户在工作时隐蔽地阅读小说。

## Requirements

### R1. 核心阅读窗口
- 无边框透明窗口，背景透明、文字不透明
- 默认置顶显示，可通过快捷键切换置顶状态
- 不在任务栏显示，通过系统托盘图标管理
- 鼠标悬停/拖拽时显示细边框，平时无边框

### R2. 显隐交互
- 鼠标移入阅读区域时显示文本，移出时隐藏
- 快捷键可"锁定"当前显示状态（鼠标移开也不隐藏），再按解锁
- 全局快捷键控制，即使窗口不在焦点也能响应

### R3. 文本滚动与导航
- 鼠标在区域内时滚轮上下滚动文本
- 自动滚动功能，可开启/关闭
- 快捷键调整自动滚动速度档位（加/减速）
- 设置界面可调整滚动速度
- 快捷键跳转上一章/下一章

### R4. 鼠标穿透
- 快捷键切换鼠标穿透模式
- 穿透模式下鼠标直接穿过阅读窗口，可操作下层窗口

### R5. 底部状态栏
- 显示当前阅读章节名称
- 显示已阅读内容占全书百分比

### R6. 文件格式支持
- 支持 .txt 格式导入
- 支持 .epub 格式导入
- 导入方式：文件选择器 + 拖拽导入

### R7. 书架与进度
- 书架功能：展示所有导入过的小说列表
- 多本小说各自独立保存阅读进度
- 切换小说时自动恢复到上次阅读位置
- 手动书签功能（可打多个书签）
- 最近阅读记录列表

### R8. 设置界面
- 独立设置窗口（从托盘菜单打开）
- 可配置项：
  - 背景透明度（滑块）
  - 字体大小（滑块或预设）
  - 字体颜色（预设 + 取色器）
  - 背景色（预设 + 取色器）
  - 行间距
  - 段间距
  - 主题/皮肤
  - 自动滚动速度
  - 快捷键自定义（所有快捷键均可重新绑定）
  - 开机自启动（默认关闭，写入注册表 HKCU\...\Run）
  - 启动时自动打开上次阅读的小说

### R9. 全局快捷键方案
| 功能 | 默认快捷键 |
|---|---|
| 显示/隐藏阅读窗口 | Ctrl+Alt+H |
| 锁定/解锁显示 | Ctrl+Alt+L |
| 开启/关闭自动滚动 | Ctrl+Alt+S |
| 滚动速度加一档 | Ctrl+Alt+Up |
| 滚动速度减一档 | Ctrl+Alt+Down |
| 上一章 | Ctrl+Alt+Left |
| 下一章 | Ctrl+Alt+Right |
| 鼠标穿透开/关 | Ctrl+Alt+P |
| 切换置顶 | Ctrl+Alt+T |
| 打开设置 | Ctrl+Alt+, |
| 打开书架 | Ctrl+Alt+B |

### R10. 数据存储
- JSON 文件存储于 `%AppData%\StealthReader\`
- 存储内容：书架列表、每本书阅读进度、书签、设置、自定义快捷键

### R11. 启动行为
- 程序启动后静默最小化到系统托盘
- 可选：启动时自动打开上次阅读的小说
- 系统托盘右键菜单：打开书架、打开设置、退出

## Acceptance Criteria

- [ ] 透明背景窗口正常显示，文字清晰可读
- [ ] 鼠标悬停显示/移出隐藏正常工作
- [ ] 锁定模式下鼠标移开文字不消失
- [ ] 滚轮翻页、自动滚动、章节跳转均正常
- [ ] 鼠标穿透模式可切换，穿透时可操作下层窗口
- [ ] .txt 和 .epub 文件均可正确导入和阅读
- [ ] 书架可展示和切换多本小说
- [ ] 多本小说进度独立保存，切换后恢复正确位置
- [ ] 所有设置项可正常调整并持久化
- [ ] 全局快捷键全部生效，可自定义
- [ ] 程序不在任务栏显示，托盘图标正常管理
- [ ] 开机自启选项正常工作

## Constraints

- 平台：Windows 10/11
- 技术栈：WPF (.NET 8+), C#
- 数据存储：JSON (System.Text.Json)
- 第三方依赖尽量少，保持轻量
